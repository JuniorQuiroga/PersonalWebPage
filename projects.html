Centro odontológico

Esta es una página web para registrar y manejar turnos tanto por parte de pacientes como médicos.
Se utilizó HTML, CSS y JS para el Frontend, Flask para el Backend encargado de gestionar inicios de sesion y manejo turnos, SQL para la base de datos y AWS para el despliegue de la página

Reproductor de música

Este es un simple reproductor de música con las capacidades que cualquier como play/pause y crear una lista de reproducción.
Se hace uso de Python en conjunto con la librería Pygame.
